java开发项目用到的技术

发表于 2018-03-05 更新于 2019-07-18 分类于 Java

Build

Tools that handle the build cycle and dependencies of an application. Apache Maven- Declarative build and dependency management that favors convention over configuration. It might be preferable to Apache Ant, which uses a rather procedural approach and can be difficult to maintain.

JSON

Libraries for serializing and deserializing JSON to and from Java objects. Gson- Serializes objects to JSON and vice versa. Good performance with on-the-fly usage.

Logging

Libraries that log the behavior of an application. SLF4J- Abstraction layer/simple logging facade.

PDF

Tools to help with PDF file creation. iText Tools to help with PDF file creation.
